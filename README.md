# ระบบ Visitor Management — สื่อการเรียนรู้ (ภาพรวมระบบ)

สื่อการเรียนรู้แบบ **interactive** อธิบายระบบบริหารจัดการผู้มาเยือน (Visitor Management System) ของหมู่บ้าน/โครงการ
ตั้งแต่ผู้ดูแลระบบสูงสุด ไปจนถึง **รปภ. ที่หน้าป้อม** ซึ่งเป็นผู้ใช้งานหลัก (App ยาม)

ออกแบบให้เรียนรู้แบบคลิกสำรวจ (แรงบันดาลใจจากสไตล์ [9arm lab](https://www.9arm.co/lab/fiberoptics.html)) — ไม่ใช่ผังนิ่ง ๆ

## 🔗 ดูออนไลน์
หลังเปิด GitHub Pages แล้ว ลิงก์จะเป็นประมาณ: `https://<username>.github.io/<repo>/`

## ▶️ ดูในเครื่อง
เป็นไฟล์ static ไฟล์เดียว ไม่ต้องติดตั้งอะไร:

- **วิธีง่ายสุด:** ดับเบิลคลิก `index.html`
- **หรือเปิดผ่าน local server:**
  ```bash
  python -m http.server 8080
  # แล้วเปิด http://localhost:8080
  ```

## 📂 โครงสร้าง
- `index.html` — ทั้งเว็บอยู่ในไฟล์เดียว (HTML + CSS + JS)
- 4 ช่วงการเรียนรู้:
  - **00 เกริ่นนำ** — ภาพรวมระบบ
  - **01 3 แกนของระบบ** — บทบาท × สถานที่ × อุปกรณ์
  - **02 ผังรวม interactive** — คลิก node เพื่อดูบทบาท/หน้าที่/อุปกรณ์ + ไฮไลต์เส้นทาง
  - **03 โฟกัส รปภ & App ยาม** — เทียบหน้าที่ป้อม 1 / ป้อม 2

## ⌨️ คีย์ลัด
- `←` `→` — เปลี่ยนหัวข้อ
- `S` — ซ่อน/แสดงเมนู
